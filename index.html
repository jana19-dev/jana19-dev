// About

I'm a software architect and entrepreneur based in Toronto, Canada. I thrive on architecting robust software systems that solve complex problems — from real-time web applications to large-scale archival platforms.

By day, I work as an Application Programmer Analyst at the University of Toronto Libraries, building and maintaining applications that serve thousands of researchers. By night, I run Codepiercer, where I freelance on dynamic web solutions — everything from static sites to high-demand applications with real-time features.

I'm a big fan of Docker, self-hosted infrastructure, serverless architecture, and clean deployments. When I'm not coding, you'll find me tinkering with DIY projects or playing board games.
